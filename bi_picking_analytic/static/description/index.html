Analytic Account on Stock Picking with Analytic Tags Odoo Apps

This Odoo apps is useful to assign analytic account and analytic tag on stock picking while you proceed the stock receipt of received goods from vendor or delivery order of goods to customer. Also This odoo apps modules pass the assigned analytic account and analytic tag automatically from Purchase order and Quotation to stock picking of receipt and delivery orders. This Odoo apps mainly use when you making manual delivery order or receipt without adding sales or purchase order so while you validate stock picking it will generate analytic account entries same as invoice and vendor bills. If you are using sales and purchase order then you will see assigned analytic account and tag automatically passed on created stock picking.

Features

Analytic Tags on Sale.

User can specify Analytic Tags on each sale order line and it will be passed in delivery order and based on that analytic entry is generated.

Analytic account on Purchase Order.

User can specify analytic account and its tags on purchase order line and pass them in incoming shipment.

Analytic Account on Sale Order

The analytic account you defined in sales order will pass in delivery order with the analytic tags. and when you done the delivery order it will create account analytic entry.

Analytic Account on Delivery Order

The sale price of the product will be recorded in account analytical line.

Analytic Account on Purchase Order

The analytic account you defined in sales order will pass in delivery order with the analytic tags. and when you done the delivery order it will create account analytic entry.

Analytic Account on Incoming Shipment

Analytical Account details are displayed in picking of particular purchase order for each products available.
