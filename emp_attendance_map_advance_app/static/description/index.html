Odoo Advance Employee Attendance Location Google Map - with Location Restriction App

This Odoo apps has very unique feature which will enhance Odoo features with this app you can register login location of the employee with google maps so that there will be no disputed attendance. It is very easy and effective way to track the employee attendance for a company. If you have sales team of multiple store system or different warehouses you can use this feature as module in Odoo ERP system. This Odoo apps also provide login/logout Location Restriction features, which only allow employee to add attendance with specific configured location.
